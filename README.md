Простое приложение для соблюдения и отслеживания еженедельных задач написанное на Dart/Flutter.
Приложение использует локальную NoSQL базу данных Isar.
В качестве дизайна использован сет виджетов Material Design.
В приложении реализованы функции:
1. Добавление задач;
2. Удаление задач;
3. Просмотр задач;
4. Поиск задач;
5. Добавление категорий (тегов) задач и присвоение их к задачам. 
Приложение «TimeManager» представляет собой инструмент для управления ежедневными задачами с акцентом на минимализм и производительность. Оно позволяет пользователям создавать, редактировать и организовывать задачи по дням недели, устанавливать время выполнения и категории. Ключевые особенности включают локальное хранение данных с использованием базы данных Isar, что обеспечивает работу без интернета, адаптивный интерфейс, соответствующий гайдлайнам Material Design, и интеграцию интерактивных элементов (FAB, диалоговые окна). Приложение не содержит рекламы и платных подписок, что делает его доступным для широкой аудитории.
